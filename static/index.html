FAQ

Isn't this illegal?
No. SpigotMC don't redistribute jarfiles due to legal issues surrounding redistributing Mojang's code, which Spigot depends on. PaperMC use a system called Paperclip that avoids this issue. These jarfiles have been "paperclipped" so they are safe to redistribute. The readme explains how it works in more detail.
Can I trust these builds?
That's for you to decide. The build script I use to put these together is open-source, and I have a good reputation on the SpigotMC forums. Naturally I encourage good security practices, don't just take my word for it, but I believe that these files are safe.
Why? What's the catch?
There's no catch. I do this because I find it fun and enjoy being helpful - the only profit I earn by running this is experience.
